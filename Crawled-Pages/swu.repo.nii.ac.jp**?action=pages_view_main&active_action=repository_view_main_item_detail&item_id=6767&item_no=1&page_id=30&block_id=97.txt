昭和女子大学学術機関リポジトリとは
	昭和女子大学学術機関リポジトリ（以下「リポジトリ」という。）は、本学の教育・研究活動を通して得られた学術成果等を、電子化により蓄積し恒久的保存を進めるとともに、学内外へ無償で公開することにより本学の学術研究の発展に寄与し、社会に貢献することを目的として構築している。
