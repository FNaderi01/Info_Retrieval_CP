  Instalar e configurar o GitHub Desktop→
Obtenha o GitHub Desktop configurado para gerenciar seu projeto de trabalho. Efetue a autenticação com GitHub.com ou GitHub Enterprise, mantenha o aplicativo atualizado e revise suas configurações de preferência.
  Contribuir e colaborar com o GitHub Desktop→
Use o GitHub Desktop para gerenciar projetos, criar commits relevantes e monitorar o histórico de cada projeto em um aplicativo em vez de usar a linha de comando.
