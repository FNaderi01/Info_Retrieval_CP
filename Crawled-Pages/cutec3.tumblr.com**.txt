Yes.

Can my computer run him?
They run as long as you install Java - which is compatible with any version of Windows starting from XP to Windows 10.
Can I use them on my stream?
Yes, if you credit me in your stream channel description with a link to this video.
Will you make more?
I am. You can support me on Patreon, I will be releasing one a month there. https://www.patreon.com/CuteC3Do I get to keep any of the Desktop Pets forever?
Yes. Once you download it it’s yours to put on any device.

Please share the discord link or this video first, as there are many instructions I’ll be posting that isn’t included in the program itself.
