        入门指南
  关于 GitHub Insights→
  GitHub Insights 提供指标和分析报告，帮助工程团队了解和改进其软件交付流程。
  GitHub Insights 系统概述→
  GitHub Insights 是一款与 GitHub Enterprise 交互的独立应用程序。
  安装 GitHub Insights→
  您可以安装 GitHub Insights 并将独立应用程序连接到 GitHub Enterprise Server。
  查看关键指标和报告→
  您可以查看和过滤关键指标和报告，帮助了解并改进通过数据交付软件的过程。
        热门文章
  关于 GitHub Insights→
  GitHub Insights 指标中包含的数据可高度自定义。
  GitHub Insights 可用的指标→
  GitHub Insights 包括各种指标，便于您了解团队的软件交付流程。
  安装和配置 GitHub Insights→
您可以通过安装和配置应用程序开始使用 GitHub Insights。
  了解 GitHub Enterprise 的使用情况→
